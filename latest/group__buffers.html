static void prepare_itbuffer ( struct secure_channel *  channel) [static]

Prepare get buffer for reception of negotiation token.

Parameters:
channelSecure channel instance for which to prepare token buffers.

During negotiation, tokens produced locally are stored into the get buffer. This function sets the pvBuffer and cbBuffer members to tell the negotiation function where to write the token. After the produced token is sent, this function should be called to clear the token buffer and discard invalidated data.

Note:
This function does not allocate allocated memory.

static void prepare_otbuffer ( struct secure_channel *  channel) [static]

Prepare put buffer for reception of negotiation token.

Parameters:
channelSecure channel instance for which to prepare token buffers.

During negotiation, tokens sent by the peer are stored into the put buffer. This function sets the cbBuffer member to 0. As data is received, token content is copied into the buffer pointed to by pvBuffer and cbBuffer is incremented to keep track of the amount of valid data. Once tokens are completely received, the negotiation function uses the data to compute a token to send to the peer in order to complete negoitation. After the token is computed, this function should be called to clear the token buffer and discard invalidated data.

Note:
This function does not allocate allocated memory.
